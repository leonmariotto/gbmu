

# GBMU : GameBoy and GameBoy Color emulator in C++

42Paris school project. <br>
The emulator can run all GameBoy MBC. <br>
Sound is WIP. <br>
Run make to compile. <br>

## Requirements
libsdl2 <br>
g++ <br>

## Joypad binding
A       -> A <br>
B       -> S <br>
Start   -> Enter <br>
Select  -> Left shift <br>
Arrows  -> Arrows <br>
